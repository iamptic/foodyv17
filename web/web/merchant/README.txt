Foody — merchant full fixed v2
1) Просто замени папку web/web/merchant/ на содержимое из архива.
2) Ничего настраивать не нужно — index.html ссылается на CSS/JS.
3) Модалка «Редактировать» по центру, поля как в «Создать оффер».
4) Удаление: 404 обрабатывается как «уже удалён».
5) Загрузка: без миганий, с защитой от гонок ответов, авто-обновление раз в 60 сек.
